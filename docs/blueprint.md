# **App Name**: AS TRUSTED CONSULTANCY

## Core Features:

- Plot Upload: Owner uploads a photo of the plot.
- Plot Viewing: Users can view the uploaded plot photos and associated details (plot no, village name, area name, plot size, plot facing).
- Owner Authentication: System ensures only the owner can upload plots and modify plot details.
- Duplicate Upload Prevention: Prevention of duplicate uploads by non-owners.
- Plot Detail Modification: Owner can change details of the plots.

## Style Guidelines:

- Earthy tones to reflect land/plots (e.g., browns, greens).
- Clear and legible font for plot descriptions (e.g., Arial, Helvetica).
- Simple, image-centric layout with easy navigation and dedicated space for plot details.
- Use icons representing location, ownership, and plot details.
